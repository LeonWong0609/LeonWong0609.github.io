用自定义 CSS 解决文章内单词词内折行问题

发布于 02-18-2020 | 分类于 博客 | 2分钟 | 490字数

最近给博客换了身新皮，叫做 Pure，主题设计得特别赞，整体简洁大方，细节也十分到位，一些小动画也特别顺畅优雅。之前的主题 Walker 存在 Title 一直不会变的问题，换上这个主题就没有发现这个问题了。然而使用过程中发现另外一个问题，英文单词会行内折行，而且也没有折行的连字符。

如下图：


* 第三行末尾的 Markd-own

上网查阅后发现可以用 CSS 来规定英文单词的换行规则，正常情况下会让单词内不折行，直接整个单词排到下一行，所以我们只在 CSS 中加上 word-break: normal; 就可以了。具体在这个主题中，对正文内容生效，也就是：

.gt-post-content {
word-break: normal;
}

我们将这段 CSS 片段加入自定义 CSS 中，效果如下：

虽然效果有点粗暴，但是起码是更加符合规范了。

前后对比图：

我们在 w3school 上可以看到更多有关 word-break 的属性，摘录部分如下：

CSS Syntax

word-break: normal|break-all|keep-all|break-word|initial|inherit;

Property Values

Value Description
normal Default value. Uses default line break rules
break-all To prevent overflow, word may be broken at any character
keep-all Word breaks should not be used for Chinese/Japanese/Korean (CJK) text. Non-CJK text behavior is the same as value "normal"
break-word To prevent overflow, word may be broken at arbitrary points
initial Sets this property to its default value. Read about initial
inherit Inherits this property from its parent element. Read about inherit
